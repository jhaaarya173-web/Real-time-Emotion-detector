# Real-time-Emotion-detector
A real-time AI-based emotion detection system using Python and OpenCV.

# Features
- Real-time face detection using webcam
- Detects emotions like:
  - Happy 
  - Sad 
  - Angry 
  - Surprise 
  - Neutral 
- Live emotion display on screen
- Voice output for detected emotions

# Technologies Used
- Python
- OpenCV
- DeepFace
- Text-to-Speech (pyttsx3)
